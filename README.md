## 简介

建立的初衷是为了知识笔记更便于管理、查看以及分享出来，然而一个人的力量是有限的，所以呢，一起来贡献自己整理的资料和知识笔记吧！

## 快速开始

### 在项目目录下安装模块

```bash
npm install
```

### 启动 Vuepress

```bash
npm run docs:dev
```
